1. Запускаємо в головній папці Bakery.sln та запускаємо проект на виконання.
Робочий порт проекту localhost:7215

2. Переходимо в папку клієнту BakeryClient та в командному рядку виконуємо команду npm i, щоб з'явилася папка node_mdules.

3. В тій же папці виконуємо команду ng serve -o, що запускає сторінку з демонстрацією.
Робочий порт проекту localhost:4200
